This challenge is about popular python framework -Django. We have to get 3 flags by repair misconfigurations in Django.

First think we have to do is login by given SSH credentials.

After open port 8000 in browser we have been informed of the need to add a local ip address to ALLOWED_HOSTS in' settings' configuration file:

After we add a properly address we could see a django admin login page but we still did not have a user, this command made it:. 


python3 manage.py createsuperuser

When we created a super user and login to the page, navigate to 'Users' and we have get first flag,

additionaly next to the flag we have a username for ssh and hashed password, after cracking hash we could easly read plaintext password.

Next step was login to ssh by given credentials and cat user.txt,and flag appeared.

Last of required flag was in the ssh server ( logged as first user) in home.html file
